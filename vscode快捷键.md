# VS Code 常用编辑快捷键备忘单

> 按 **Windows / Linux** 为主，括号内为 **Mac** 对应按键。

---

## ✍️ 通用编辑与基础操作

| 功能 | Windows / Linux | Mac |
| :--- | :--- | :--- |
| 复制 | `Ctrl + C` | `Command + C` |
| 剪切 | `Ctrl + X` | `Command + X` |
| 粘贴 | `Ctrl + V` | `Command + V` |
| 撤销 | `Ctrl + Z` | `Command + Z` |
| 重做 | `Ctrl + Y` | `Command + Shift + Z` |
| 全选 | `Ctrl + A` | `Command + A` |

---

## 🎯 光标移动与文本选择

| 功能 | Windows / Linux | Mac |
| :--- | :--- | :--- |
| 移动光标到行首 / 行尾 | `Home` / `End` | `Command + ←` / `Command + →` |
| 移动光标到文件头 / 文件尾 | `Ctrl + Home` / `Ctrl + End` | `Command + ↑` / `Command + ↓` |
| 按单词移动光标 | `Ctrl + ←` / `Ctrl + →` | `Option + ←` / `Option + →` |
| 从光标处选择到行首 / 行尾 | `Shift + Home` / `Shift + End` | `Shift + Command + ←` / `Shift + Command + →` |
| 按单词选择文本 | `Ctrl + Shift + ←` / `Ctrl + Shift + →` | `Option + Shift + ←` / `Option + Shift + →` |

## 📝 行操作与代码编辑

| 功能 | Windows / Linux | Mac |
| :--- | :--- | :--- |
| 向上 / 下移动当前行 | `Alt + ↑` / `Alt + ↓` | `Option + ↑` / `Option + ↓` |
| 向上 / 下复制当前行 | `Shift + Alt + ↑` / `Shift + Alt + ↓` | `Shift + Option + ↑` / `Shift + Option + ↓` |
| 删除当前行 | `Ctrl + Shift + K` | `Command + Shift + K` |
| 在当前行上方 / 下方插入空行 | `Ctrl + Shift + Enter` / `Ctrl + Enter` | `Command + Shift + Enter` / `Command + Enter` |
| 缩进 / 取消缩进 | `Tab` / `Shift + Tab` | `Tab` / `Shift + Tab` |
| 切换行注释 | `Ctrl + /` | `Command + /` |
| 添加 / 移除块注释 | `Shift + Alt + A` | `Shift + Option + A` |
| 代码格式化 | `Shift + Alt + F` | `Shift + Option + F` |

## ⚙️ 自定义快捷键

如果默认快捷键不生效或想按自己习惯修改，可以按：

- **Windows / Linux**：`Ctrl + K, Ctrl + S`
- **Mac**：`Command + K, Command + S`

打开快捷键设置界面，搜索并修改任意快捷键。
